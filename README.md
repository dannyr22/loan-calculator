# Loan Calculator

A React Native app built with Expo for calculating loan repayments.

## Prerequisites

- [Node.js](https://nodejs.org/) 20.19.4 — managed via [nvm](https://github.com/nvm-sh/nvm)
- iOS Simulator or Android Emulator

## Setup

```bash
nvm use
yarn install
```

## Running

```bash
yarn start
```

Press `i` for iOS, `a` for Android, or scan the QR code with [Expo Go](https://expo.dev/go).

## Testing

```bash
yarn test
```

## Common Issues

**Metro cache**
```bash
npx expo start --clear
```

**Port in use**
```bash
npx expo start --port 8082
```
